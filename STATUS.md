# 🛰️ LeadOS — статус проекта

> Живой документ. Обновляется по ходу. Последнее обновление: 2026-06-22.

## Суть в одном предложении
Радар, который находит в интернете людей, **публично пишущих, что им нужно то, что решает твой продукт**, оценивает «живой ли это запрос», пишет человечный ответ и утром присылает в Telegram. Оружие фаундера для тихого поиска клиентов без рекламы и спама.

## Где живёт
- **Живой SaaS:** https://signalos-es52.onrender.com (Render + Supabase Postgres)
- **Telegram-бот доставки:** @LeadOS_AI_Bot
- **Репозиторий:** github.com/MASTEROCH/signalos (тех-имя пока старое)
- **Локальный код:** `~/code/signalos/` · запуск `./run.sh` (SQLite)

## Стадия
**Живой рабочий быстрый MVP.** Ядро (поиск → резонанс → ответ → дайджест) построено и проверено на проде. Главный узкий момент — **охват каналов**: богатые источники (Reddit, соцсети, VK) надо разблокировать ключами.

## ✅ Что УМЕЕТ (построено и проверено)
- Аккаунты: регистрация/вход, изоляция данных, кабинет
- 8 каналов: HackerNews, Reddit, Bluesky, Lemmy, GitHub, Stack Exchange, Mastodon, Google Alerts/RSS
- Резонанс-скоринг: живой запрос vs сухое упоминание + топикальность (по теме ли продукта)
- 8 готовых пресетов под продукты Роча (ключи + резонанс + каналы в один тап)
- Каналы под проект (SELFIX → соцсети, AppHub → GitHub, и т.д.)
- Фильтр свежести (3/7/14/30 дней)
- Готовые ответы (Claude / свой ключ) + спам-чек + UTM
- Утренний Telegram-дайджест (тап-копи ответа)
- Быстрый: сканы 4–8 сек, кэш + пул соединений + параллельные источники
- Экспорт лидов в CSV, ИИ-улучшение поиска

## ❌ Чего НЕ умеет / ограничения (честно)
- **Reddit / Bluesky / Mastodon заблокированы с сервера** (датацентр-IP) → Reddit оживёт с бесплатными API-ключами
- **Сейчас реально работают с сервера:** HackerNews, Lemmy, GitHub — технари/англоязычные → тонко для RU/локальной аудитории
- **Нет Instagram / Facebook / Threads / LinkedIn** — нет публичного API для прослушки (стены). Заходим через Google-поиск (`site:`)
- **Нет VK** (в работе — открытый API, главный RU-выигрыш)
- **Нет X/Twitter** (платный API или обход через Google-поиск)
- **Оплата — заглушка** (реальные деньги пока нельзя; Cryptomus в планах)
- **Инфра:** Render (США) ↔ Supabase (Франкфурт) — кэш маскирует; free-тариф спит (~50 сек холодный старт)
- Качество ответов заметно лучше с активным Claude-ключом (free-режим грубее)
